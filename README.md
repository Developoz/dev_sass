# gulp_auto

Requirements

Install Node

Prior to taking the course, clone this repo and run the content locally

$ npm install
$ gulp

  |- app/
      |- css/
      |- images/ 
      |- index.html
      |- js/ 
      |- sass/
  |- dist/
  |- gulpfile.js
  |- node_modules/
  |- package.json
  
  Type in gulp in the command line and you're good to go.
