# dev_sass
First make sure to install gulp $ sudo npm install gulp -g

Clone the document and use $ npm install

this is how the path structure looks like

  |- app/
      |- css/
      |- images/ 
      |- index.html
      |- js/ 
      |- sass/
  |- dist/
  |- gulpfile.js
  |- node_modules/
  |- package.json
  
  Type in gulp in the command line and you're good to go.
